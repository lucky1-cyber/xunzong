# 非遗寻踪 - 开放世界游戏项目

## 项目说明
这是一个符合实验2要求的PC端网页布局项目，主题为"非遗寻踪"开放世界游戏。通过WebGL 3D渲染展示中国非物质文化遗产。

## 项目主题

**《非遗寻踪》**

### 选题目标

- **用户目标**: 为年轻网民（16-30岁）提供一个像"开放世界游戏"一样有趣的非遗探索体验，降低传统文化认知门槛
- **技术目标**: 深度融合 WebGL 3D 渲染、物理引擎、空间音频与动态叙事技术，构建一个高性能的浏览器内沉浸式环境
- **价值目标**: 将静态的"非遗知识"转化为动态的"文化体验"，激发年轻一代对传统文化的主动探索欲和认同感

## 完成的功能

### ✅ 实验2要求已完成：

1. **启动页布局 (index.html)**
   - 顶部导航栏：固定定位
   - 中部核心视觉区：70%高度，展示项目介绍
   - 底部功能入口：20%高度，两个按钮

2. **三维地图场景页 (map.html)**
   - 左侧地图区：50%宽度，中国地图+非遗标记点
   - 右侧内容区：50%宽度，非遗项目介绍和任务

3. **页面链接**
   - 首页可以跳转到地图场景页
   - 地图页可以返回首页
   - 导航栏菜单实现页面跳转

4. **CSS布局技术**
   - 使用百分比布局实现高度分配
   - 使用Flexbox实现元素居中和对齐
   - 使用固定定位实现导航栏

## 使用方法

1. 直接在浏览器中打开 `index.html` 查看首页
2. 点击"开始探索"按钮或导航栏的"三维地图场景"进入地图页
3. 在地图页点击"返回首页"或导航栏的"启动页"返回

## 核心交互逻辑（页面展示）

1. **探索**: 支持键盘（WASD）漫游和点击地图图标快速旅行两种模式
2. **发现**: 玩家角色靠近可交互物体时，物体会高亮，并出现提示图标
3. **学习与互动**: 通过简单的交互（如点击、拖拽、长按）完成小任务

## 特色交互设计（概念展示）

- **虚拟实操**: 使用鼠标拖拽模拟"拉坯"动作，物理引擎提供真实的力度反馈
- **动态叙事**: 非遗传承人以音频导游的形式出现，根据玩家的进度和位置触发不同的讲解
- **AR融合**: 在特定点位，可调用摄像头，将虚拟的"青花瓷"叠加到用户的真实桌面上进行合影并分享
- **游戏化激励**: 完成一个项目的体验后，解锁对应的"非遗守护者"成就徽章，并生成一张包含AR合照和成就的分享海报

## 设计特色

### 视觉设计
- **色彩方案**: 古铜金渐变（#d4a574 → #8b6f47），体现文化底蕴
- **背景色**: 米黄色（#f4ede3），营造纸质古籍感
- **动画效果**: 文物标记点浮动、罗盘旋转

### 布局特点
- **首页**: 采用"完整显示(contain)"方式
- **地图页**: 左右分栏，便于地图与内容对比查看
- **导航栏**: 固定定位，始终可见

## 技术栈

- HTML5
- CSS3 (Flexbox、动画、渐变、SVG)
- 预留WebGL 3D渲染接口

## 注意事项

- 本项目为静态页面，无JavaScript交互（符合实验2要求）
- 地图为SVG简化版本，可替换为实际图片或WebGL场景
- images文件夹预留，可添加背景图和非遗文物图片
- 符合实验2的所有布局要求
